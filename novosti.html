Naučite HTML

HTML je kratica za HyperText Markup Language, što znači prezentacijski jezik za izradu web stranica. Hipertekst dokument stvara se pomoću HTML jezika. HTML jezikom oblikuje se sadržaj i stvaraju se hiperveze hipertext dokumenta. HTML je jednostavan za uporabu i lako se uči, što je jedan od razloga njegove opće prihvaćenosti i popularnosti.

Svoju raširenost zahvaljuje jednostavnosti i tome što je od početka bio zamišljen kao besplatan i tako dostupan svima. Prikaz hipertekst dokumenta omogućuje web preglednik. Temeljna zadaća HTML jezika jest uputiti web preglednik kako prikazati hipertext dokument.

Pri tome se nastoji da taj dokument izgleda jednako bez obzira o kojemu je web pregledniku, računalu i operacijskom sustavu riječ. HTML nije programski jezik niti su ljudi koji ga koriste programeri. Njime ne možemo izvršiti nikakvu zadaću, pa čak ni najjednostavniju operaciju zbrajanja ili oduzimanja dvaju cijelih brojeva. On služi samo za opis naših hipertekstualnih dokumenata. Html datoteke su zapravo obične tekstualne datoteke, ekstenzija im je .html ili .htm.

Osnove CSS-a

CSS je kratica od (eng.) Cascading Style Sheets. Radi se stilskom jeziku, koji se koristi za opis prezentacije dokumenta napisanog pomoću markup (HTML) jezika.

Deklaracijski blok su vitičaste zagrade unutar kojih se nalaze deklaracije. Svaka deklaracija sastoji se od svojstva, dvotačke (:) i vrijednosti. Između svake dvije uzastopne deklaracije mora se nalaziti tačka zarez (;).

Vrijednosti mogu biti ključne riječi poput "center" (sredina) i "inherit" (naslijedi), brojčane vrijednosti poput 100 (debljina fonta), 200px (200 piksela), 50vw (50 % širine viewporta) ili 80% (80 % širine prozora).

Vrijednosti boja mogu biti ključne riječi (npr. "red" za crveno), heksadecimalne vrijednosti (npr. #FF0000 ili #F00), RGB vrijednosti od 0 do 255 (npr. rgb(255, 0, 0)), RGBA vrijednosti koje uključuju i alpha prozirnost (npr. rgba(255, 0, 0, 0.8)) ili HSL/HSLA vrijednosti (npr. hsl(000, 100%, 50%), hsla(000, 100%, 50%, 80%)).

Uvod u JS

JavaScript je skriptni programski jezik, koji se izvršava u web pregledniku na strani korisnika. Napravljen je da bude sličan Javi, zbog lakšeg korištenja, ali nije objektno orijentisan kao Java, već se temelji na prototipu i tu prestaje svaka povezanost s programskim jezikom Java.

Uz HTML i CSS, JavaScript je jedna od glavnih tehnologija "World Wide Web-a". JavaScript omogućava interaktivne web stranice i bitan je dio web aplikacija. Velika većina web stranica koristi ga, a glavni web preglednici imaju namjenski JavaScript mehanizam za njegovo izvršavanje.

Iako postoje sličnosti između JavaScripta i Jave, uključujući naziv jezika, sintaksu i odgovarajuće standardne bibioteke, dva su jezika različita i vrlo se razlikuju po dizajnu. Na JavaScript su utjecali programski jezici kao što su Self i Scheme. Format serializacije JSON, koji se koristi za pohranjivanje podatkovnih struktura u datoteke ili za prijenos putem mreže, temelji se na JavaScriptu.

HTML meta tagovi

Meta tagovi su dijelovi HTML-a u stranici koje upotrebljavaju tražilice da bi zapisale informacije o vašoj stranici. Ovi tagovi sadrže ključne riječi, opis, informaciju o vlasništvu, naziv stranice itd.

Oni su među mnogim stvarima koje ispituju tražilice kada "gledaju" vašu stranicu. Iako ih nije nužno, vrlo ih je korisno upotrebljavati, pogotovo ako želite imati dobru poziciju na tražilicama (a tko ne želi?).

Ako napravite web stranicu i registrirate URL kod tražilica, one će posjetiti vašu stranicu i pokušati je indexirati. Svaka tražilica funkcionira malo drugačije, i svaka drukčije cijeni pojedine elemente web stranice.

Meta tagovi bi uvijek trebali biti smješteni u head područje HTML dokumenta. head tag se nalazi odmah nakon html taga, i završava prije body taga.

Primjer:
<title> Naziv stranice </title>
<meta name="description" content="opis web stranice">
<meta name="keywords" content="popis ključnih riječi vezanih za sadržaj stranice">
<meta name="robots" content="index,follow">

CSS varijable

Prilagođena svojstva (koja se ponekad nazivaju i CSS varijable ili kaskadne varijable) su entiteti definirani od strane CSS autora koji sadrže određene vrijednosti koje će se ponovo koristiti u dokumentu. Postavljaju se pomoću prilagođene osobine svojstava (npr. --glavna-boja: crna;) i pristupa im se funkcijom var () (npr. color: var (--glavna-boja);).

Složene web stranice sadrže vrlo velike količine CSS-a, često s puno ponovljenih vrijednosti. Na primjer, ista se boja može koristiti na stotinama različitih mjesta, što zahtijeva globalno pretraživanje i zamjenu ako se ta boja mora promijeniti.

Upotreba

Deklaracija prilagođenog svojstva:

element{
--glavna-boja: #f3b4a2;
}

Korištenje prilagođenog svojstva:

element{
color: --glavna-boja;
}

JS sintaksa

U JavaScript-u, varijable se mogu definisati koristeći ključne riječi let, var ili const:

let x; // deklariše varijablu i dodjeljuje joj vrijednost "undefined"
var y = 2; // deklariše varijabli dodjeljuje joj brojčanu vrijednost 2
const z ="Hello, World!" // deklariše varijablu i dodjeljuje joj string vrijednost "Hello, World!";

Komentari, kao u gornjem primjeru se stavljaju nakon znakova //.

JavaScript nema ugrađenu funkcionalnost za upis/ispis, nego to omogućava run-time okruženje. Primjer:

console.log("Hello, World!");

Primjer jednostavne rekurzivne funkcije:

function factorial(n){
if(n === 0){
return 1;
}
return n*factorial(n-1);
}
factorial(3); // vraća vrijednost 6

Slike u HTML-u

Kada bi web stranice sadržavale samo tekst i dalje bi bile tehnološki impresivne, ali bez slika ne bi bile niti približno zabavne kao danas. Dva najčešće upotrebljavana grafička formata datoteka na internetu su GIF i JPEG

JPEG (Joint Photographic Experts Group) format primarno se koristi za prikaz realističnih slika fotografske kvalitete, dok se gif koristi za slike s manjim spektrom boja, kao što su na primjer navigacijske ikone, banneri, animacije itd.

Među web dizajnerima veliku je popularnost stekao i format PNG koji je stvoren kako bi poboljšao i zamjenio GIF format. PNG podržava slike zasnovane na paleti (s paletom definiranom 24 bitnim RGB bojama), greyscale slike (sivih tonova) i RGB slike.

Primjer:

<html>
<head>
<title>Primjer ubacivanja slike</title>
</head>
<body>
<img src="link_lokacije_slike.jpg"/>
<body/>
<html>

CSS selektori

CSS Selektor označava dio markupa na koji se primjenjuje stil. Od specifičnosti CSS selektora zavisi koji stil će biti primijenjen na odabrani HTML element. Selektor može biti:

  • svi elementi istog tipa, npr. svi h2 elementi
  • elementi određenog id ili class atributa:
    • id: jedinstven element
    • class: može obuhvatiti više od jednog elementa
  • elementi u odnosu na druge elemente u DOM-u

Pseudoklase su klase koje omogućuju opisivanje informacija koje nisu dostupne u DOM-u poput :hover koji identificira sadržaj samo ako korisnik drži pokazivač nad sadržajem..

Upotreba JavaScript-a u web stranicama

Od maja 2017, 94.5% od 10 miliona najpopularnijih web stranica koristilo je JavaScript. Najčešća upotreba JavaScript-a je dodavanje ponašanja na strani klijenta na HTML stranice, također poznato kao dinamički HTML (DHTML).

Skripte su ugrađene ili uključene s HTML stranica i međusobno su povezane s modelom objekta dokumenta (DOM) stranice.

Neki jednostavni primjeri upotrebe su:

  • Učitavanje sadržaja nove stranice ili slanje podataka na poslužitelj putem Ajaxa bez ponovnog učitavanja stranice (na primjer, društvena mreža može omogućiti korisniku da objavljuje statuse bez napuštanja stranice).
  • Animacija elemenata stranica, njihovo blijeđenje i smanjivanje, mijenjanje veličine, pomicanje itd.
  • Interaktivni sadržaj, naprimjer igre te audio i video zapisi.
  • Provjeravanje ulaznih vrijednosti web obrasca da biste bili sigurni da su prihvatljive prije slanja na poslužitelj.
  • Prijenos informacija o navikama čitanja korisnika i aktivnostima pregledavanja na razne web stranice. Web stranice to često rade za web analitiku, praćenje oglasa, personalizaciju ili druge svrhe.
